# ATM Interface

## Description
This project is a Java console-based ATM Interface developed as part of the Oasis Infobyte Internship. It simulates basic banking operations such as login, deposit, withdrawal, fund transfer, and transaction history.

## Features
- Secure login using User ID and PIN
- Deposit money
- Withdraw money
- Transfer funds between accounts
- Transaction history
- Balance checking
- Invalid login attempt handling

## Technologies Used
- Java
- OOP Concepts
- ArrayList
- HashMap
- Scanner

## Project Structure
- Account.java
- Transaction.java
- Bank.java
- ATM.java
- Main.java

## How to Run

Compile all Java files:

```bash
javac *.java
```

Run the project:

```bash
java Main
```
